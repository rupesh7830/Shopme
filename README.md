# ShopMe

A modern E-Commerce web application built with Laravel.

## Features

- User Authentication
- Product Management
- Category Management
- Shopping Cart
- Wishlist
- Responsive Design

## Tech Stack

- Laravel 12
- PHP
- MySQL
- Tailwind CSS
- JavaScript

## Installation

git clone ...
composer install
npm install
php artisan migrate
